mat-grid-list is a two-dimensional list view that arranges cells into grid-based layout. See Material Design.
An mat-grid-list must specify a cols attribute which sets the number of columns in the grid.
The number of rows will be automatically determined based on the number of columns and the number of items.
rowHeight : The height of the rows in a grid list can be set via the rowHeight attribute
 1:> Fixed height: The height can be in px, em, or rem. If no units are specified, px units are assumed (e.g. 100px, 5em, 250
 2:> Ratio: This ratio is column-width:row-height, and must be passed in with a colon, not a decimal (e.g. 4:3).
 2:> Fit: Setting rowHeight to fit This mode automatically divides the available height by the number of rows. Please note the height of the grid-list or its container must be set.
Note: If rowHeight is not specified, it defaults to a 1:1 ratio of width:height. mat-divider : is used fro the partion in between the list
