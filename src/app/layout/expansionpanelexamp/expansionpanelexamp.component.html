mat-expansion-panel provides an expandable details-summary view.
Each expansion-panel must include a header and may optionally include an action bar..
The number of rows will be automatically determined based on the number of columns and the number of items.
Header : mat-expansion-panel-header shows a summary of the panel content and acts as the control for expanding and collapsing
  This header may optionally contain an mat-panel-title and an mat-panel-description which format the content of the header to align with Material Design specifications.
Action bar : Actions may optionally be included at the bottom of the panel, visible only when the expansion is in its expanded state.
Disabling a panel : Expansion panels can be disabled using the disabled attribute..
