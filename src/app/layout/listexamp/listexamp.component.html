mat-list is a container component that wraps and formats a series of line items.
As the base list component, it provides Material Design styling, but no behavior of its own..
mat-list element contains a number of mat-list-item elements
Use mat-nav-list tags for navigation lists (i.e. lists that have anchor tags).
Use the mat-action-list element when each item in the list performs some action. Each item in an action list is a button element.
mat-list-option : A selection list provides an interface for selecting values, where each list item is an option.
mat-divider : is used fro the partion in between the list
